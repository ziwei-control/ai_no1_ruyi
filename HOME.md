## 📊 技术亮点

### 🧩 插件化架构
```
NLP层 → Path层 → Exec层 → Report层 → Evolve层
```

### 🛡️ 熔断机制
- 连续失败 > 3 → 自动熔断跳过
- 任务成功 → 重置计数

### 🔄 循环控制
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--次数=N` | 最多执行 N 轮 | 19 |
| `--超时=N` | 单次超时 N 秒 | 120 |
| `--收敛=N` | 连续 N 次相同结果则停 | 已收敛即停 |
| `--直达=目标` | 达成目标即停止 | 无 |
