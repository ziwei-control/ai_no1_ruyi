# 🚀 OpenClaw-bot-review 快速安装指南

## ⚡ 5 分钟快速开始

### 前提条件

- ✅ Node.js 18+ 
- ✅ npm 包管理器
- ✅ OpenClaw 已安装（可选，用于连接真实数据）

---

## 📋 步骤 1: 检查环境

```bash
# 检查 Node.js 版本
node --version
# 需要 v18 或更高

# 检查 npm
npm --version
```

**如果 Node.js 版本太低：**

```bash
# 安装 Node.js 18
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
```

---

## 📋 步骤 2: 安装项目

```bash
# 进入 CoPaw 目录
cd /root/.copaw

# 克隆项目（如果网络不好，多试几次）
git clone https://github.com/xmanrui/OpenClaw-bot-review.git

# 进入项目目录
cd OpenClaw-bot-review

# 安装依赖
npm install
```

**预计时间：** 3-5 分钟

---

## 📋 步骤 3: 配置（可选）

**如果不连接 OpenClaw，跳过此步直接启动即可。**

**如果需要连接 OpenClaw：**

```bash
# 检查 OpenClaw 配置
ls -la ~/.openclaw/openclaw.json

# 如果没有，需要先安装 OpenClaw
npm install -g openclaw
openclaw onboard
```

**自定义配置（可选）：**

```bash
# 创建 .env 文件
cat > .env << EOF
OPENCLAW_HOME=/root/.openclaw
PORT=3000
EOF
```

---

## 📋 步骤 4: 启动 Dashboard

```bash
# 开发模式
npm run dev
```

**看到以下输出表示成功：**

```
> openclaw-bot-review@1.0.0 dev
> next dev

  ▲ Next.js 14.x.x
  - Local:        http://localhost:3000
  - Ready in xxx ms
```

---

## 📋 步骤 5: 访问 Dashboard

**打开浏览器访问：**

## 🔧 常用命令

### 启动/停止

```bash
# 启动（前台）
npm run dev

# 启动（后台）
nohup npm run dev > dashboard.log 2>&1 &

# 停止
pkill -f "npm run dev"

# 查看进程
ps aux | grep "npm run dev"
```

### 查看日志

```bash
# 查看输出日志
tail -f dashboard.log

# 查看 Next.js 日志
# 在前台运行时直接显示
```

### 重启

```bash
# 停止并重启
pkill -f "npm run dev"
npm run dev
```

## 🐛 常见问题

### 问题 1: git clone 超时

**解决：**
```bash
# 使用 GitHub 镜像
git clone https://ghproxy.net/https://github.com/xmanrui/OpenClaw-bot-review.git

# 或手动下载 ZIP
wget https://github.com/xmanrui/OpenClaw-bot-review/archive/refs/heads/main.zip
unzip main.zip
mv OpenClaw-bot-review-main OpenClaw-bot-review
```

---

### 问题 2: npm install 失败

**解决：**
```bash
# 使用淘宝镜像
npm install --registry=https://registry.npmmirror.com

# 或清除缓存重试
npm cache clean --force
npm install
```

---

### 问题 3: 端口 3000 被占用

**解决：**
```bash
# 查找占用端口的进程
lsof -i :3000

# 杀死进程
kill -9 <PID>

# 或修改端口
PORT=3001 npm run dev
```

---

### 问题 4: 像素办公室不显示

**解决：**
```bash
# 清除 Next.js 缓存
rm -rf .next

# 重新启动
npm run dev
```

---

### 问题 5: 无法连接 OpenClaw

**解决：**
```bash
# 检查 OpenClaw 是否运行
openclaw gateway status

# 启动 Gateway
openclaw gateway run

# 检查配置
cat ~/.openclaw/openclaw.json
```

## 📊 下一步

### 1. 配置 CoPaw 管理技能

创建 `/root/.copaw/active_skills/dashboard_manager/SKILL.md`：

```markdown
# Dashboard Manager

## 功能
- 启动/停止 Dashboard
- 检查状态
- 查看统计

## 使用
用户：启动 Dashboard
助手：正在启动...
```

### 2. 添加自动启动

```bash
# 创建 systemd 服务
sudo nano /etc/systemd/system/openclaw-dashboard.service

# 内容：
[Unit]
Description=OpenClaw Dashboard
After=network.target

[Service]
WorkingDirectory=/root/.copaw/OpenClaw-bot-review
ExecStart=/usr/bin/npm run dev
Restart=always

[Install]
WantedBy=multi-user.target

# 启用
sudo systemctl enable openclaw-dashboard
sudo systemctl start openclaw-dashboard
```

### 3. 配置远程访问

```bash
# 使用 Nginx 反向代理
sudo nano /etc/nginx/sites-available/dashboard

# 内容：
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
    }
}

# 启用
sudo ln -s /etc/nginx/sites-available/dashboard /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```
