# 🤖 智能一号（AI No.1）Skill

世界顶级的自动化测试架构师技能。它能对任意工具/命令/脚本进行深度分析 → 用例设计 → 执行测试 → 问题诊断 → 报告生成 + 新 Skill 自动生成。

### 🎯 核心能力
- **深度分析**：理解工具功能，识别关键功能点、边界条件、异常路径  
- **智能用例生成**：设计 5~15 个用例（含正常/异常/边界场景）  
- **自动执行**：调用 `execute_shell_command` 或其他 tools 测试真实行为  
- **根因诊断**：通过返回码、stdout/stderr、日志模式匹配，定位：  
  - 环境问题（路径、权限、依赖缺失）  
  - 参数问题（类型错误、格式不匹配）  
  - 逻辑 bug（结果偏离预期）  
- **报告与进化**：生成 Markdown 测试报告 + 自动编写新 Skill 代码文件（`skills/<tool>/SKILL.md` + `tools.yml`）

### 🛠️ 使用方式
你只需输入：
```
测试 curl：下载网页并验证返回 200 状态码
测试 git：初始化仓库并提交一个文件
测试 ffmpeg：把 PNG 转成 MP4
```

**智能一号** 将自动执行全闭环，并生成：
1. `/root/.copaw/memory/YYYY-MM-DD.md` 记录测试事件  
2. `/root/.copaw/skills/<tool>/SKILL.md` 新技能文件（可立即使用）

### 🚀 典型工作流
1. **分析** → 提取关键功能、参数、输出格式  
2. **设计用例** → 至少 5 个测试场景（含 2+ 异常）  
3. **执行** → 调用系统命令/工具  
4. **诊断** → 比对预期 vs 实际，定位 failure 类型  
5. **报告** → 结构化 Markdown  
6. **进化** → 生成修复后的新 Skill（含错误处理、重试逻辑）

### 📝 输出格式
- 报告：`AI_No1_Report_<tool>_<timestamp>.md`  
- 新 Skill：`/root/.copaw/skills/<tool>/SKILL.md`
