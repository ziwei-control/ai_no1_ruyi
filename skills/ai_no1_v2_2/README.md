| 缺失项 | 问题 |
|--------|------|
| **路径/命令** | “请提供要测试的脚本或工具名（如 `/path/to/script.sh` 或 `curl`）” |
| **循环次数** | “需要测试几次？默认 1 次（支持 `--次数=3` 格式）” |

| **Docker 镜像** | “Docker 镜像名是？（如 `nginx:latest`）” |
