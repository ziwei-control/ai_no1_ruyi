# 🤖 智能一号 v2.0（AI No.1 v2.0）· 快速入门指南

## 🚀 指令格式（超简单）

```
测试 <工具>：[参数]
```

### 可选参数

| 参数 | 示例 | 说明 |
|------|------|------|
| `--次数=N` | `--次数=3` | 最多执行 N 轮（默认 1） |
| `--直达=目标` | `--直达=200` | 达到目标即停止（如 HTTP 200） |
| `--超时=N` | `--超时=120` | 单次测试超时 N 秒（默认 60） |
| `--收敛=N` | `--收敛=2` | 连续 N 次相同结果则停止 |

---

## 💡 示例

### 1. 次数驱动（最多 3 轮）
```
测试 jq：--次数=3
```
→ 执行 1~3 轮，达到 3 次即退出

---

### 2. 结果导向（直到成功）
```
测试 wget：--直达=200 --次数=5
```
→ 循环测试，直到 HTTP 状态为 200，或执行 5 次后退出

---

### 3. 超时保护
```
测试 ffmpeg：--超时=300
```
→ 单次测试最长 5 分钟，防止卡死

---

### 4. 收敛检测（稳定了就停）
```
测试 curl：--收敛=2
```
→ 如果连续 2 轮结果相同，则停止测试（认为稳定）

## 🧠 使用注意

- `--次数` 和 `--收敛` 可组合（如 `--次数=5 --收敛=2`）  
  → 每轮都检查收敛，**任一条件满足即停**
- `--直达=成功` = `exit=0`，`--直达=失败` = `exit≠0`
- 不加参数 = 默认 `--次数=1`（v1.0 行为）
