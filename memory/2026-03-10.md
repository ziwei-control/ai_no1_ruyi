## 📋 邮件找回个人密钥需求（筹备中）

### 🔑 基本流程
1. 用户在登录页点击「忘记密钥？」  
2. 输入注册邮箱  
3. 系统发送 6 位验证码到邮箱  
4. 用户输入验证码 + 新密钥  
5. 系统验证通过后更新加密数据  
6. 成功后跳转登录页

### 📝 潜在实现计划
| 模块 | 文件 |
|------|------|
| 路由 | `routes/forgot.js` |
| 视图 | `views/forgot.ejs` / `views/reset-password.ejs` |
| 数据模型 | `models/EncryptedUser.js`（加 `verifyCode`/`verifyCodeExpires`/`verifyCodeTries` 字段） |
| 邮件 | `config/email.js`（加 `sendVerifyCode` 函数） |
| 路由注册 | `routes/index.js`（加 `/forgot` + `/reset-password`） |
