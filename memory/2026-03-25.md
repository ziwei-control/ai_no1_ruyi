# 2026-03-25 PandaCo 音乐系统开发日志

## 🎯 主要任务
优化 PandaCo 音乐上传系统，实现视频/音频上传、自动转换、AI 智能命名、双库同步功能。

## ✅ 完成的功能

### 1. 音乐上传功能优化
- **支持格式**：
  - 音频：MP3, WAV, OGG, FLAC, M4A, AAC, WMA, APE, OPUS
  - 视频：MP4, AVI, MKV, MOV, WMV, FLV, WebM, M4V, MPEG, MPG, 3GP
- **文件大小限制**：100MB
- **自动转换**：视频文件自动提取音频并转换为 MP3（192kbps, 44.1kHz）
- **音频处理**：音频文件保持原始格式保存

### 2. AI 智能命名功能
根据音频时长自动生成优雅的音乐名称：
- < 30 秒 → "晨曦微光" - 清风乐团
- 30-60 秒 → "静谧时光" - 流年音乐社
- 1-2 分钟 → "心灵絮语" - 梦境演奏家
- > 2 分钟 → "岁月如歌" - 时光留声机

支持用户自定义标题和艺术家，AI 作为默认值。

### 3. 双库同步功能
- 自动同步到 GitHub 和 Gitee 仓库
- 仓库路径：`/root/music-repo/`
- 同步文件：音乐文件 + music_list.json
- 异步执行，不阻塞上传响应

### 4. 格式转换独立功能
- 在音乐页面添加"🔄 格式转换"按钮
- 专门用于视频转 MP3
- 与上传音乐功能分离（后续又合并）

### 5. 文件保存修复
**问题**：转换后的文件保存在 temp 目录而非 uploads 目录
**解决**：修改代码确保转换后的 MP3 文件直接保存到 `/root/eat/public/music/uploads/`

### 6. 过期音乐清理
- 用户上传的音乐 24 小时后自动过期
- 定期清理过期文件和记录
